Небольшая программа для использования шифра Виженера.

Описание
Программ позволяет шифровать или расшифровывать данные по ключу. Сохранять полученный результат
на жесткий диск. Программа работает только с кириллицей в нижнем регистре, остальные символы
не обрабатываются и остаются без изменений.

Основные функции:

1.Открыть файл в формате .txt
2.Расшифровать данные из файла зашифрованные с помощью шифра Виженера(по введенному ключу)
3.Сохранить результат дешифровки в новый .txt файл с указанием названия и директории
4.Возможность указать путь вручную или выбрать в окне
5.Краткий просмотр информации об основных функциях 
6.Зашифровать сообщение с помощью шифра Виженера(по введеному ключу)
7.Сохранить результат шифрования в новый .txt файл с указанием названия и директории
8.Расшифровать/Зашифровать текст, введённый в окно программы, и сохранить результат

Так же в программе реализовано несколько и Unit тестов для проверки работы основных функций
